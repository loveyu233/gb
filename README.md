# GB 后端开发框架

GB 是一个基于 Gin 框架的 Go 语言后端开发框架，提供了丰富的功能和工具，帮助开发者快速构建高性能的 Web 应用。

## 主要特性

- 🚀 基于 Gin 框架的 HTTP 服务器
- 💰 快捷启动支付宝和微信支付已经对应小程序登录
- 🔐 JWT 认证支持
- 📝 自动生成 Swagger API 文档
- 🎯 优雅的路由管理
- ⏱️ 请求追踪和日志记录
- 🔄 优雅关闭支持
- 📊 GORM Redis ETCD数据库集成
- 🔍 参数验证（支持中文错误提示）
- 🕒 时间处理工具（支持中国时区）
- 💰 Decimal 处理工具
- 🚨 统一的错误处理机制
- 📝 结构化的日志记录
- 🧰 快速使用(ants,gjson,copier,lo)
